Sistem Manajemen Stok Toko Elektronik

Deskripsi:
Sistem Manajemen Stok Toko Elektronik adalah aplikasi berbasis Java yang digunakan untuk mengelola data produk elektronik. Program ini dibuat sebagai tugas mata kuliah Algoritma dan Struktur Data dengan menerapkan konsep OOP, Array, Searching, Sorting, dan File Handling tanpa menggunakan Collection Framework.

Fitur:
CRUD:
- Tambah Produk
- Tampilkan Semua Produk
- Edit Produk
- Hapus Produk (Soft Delete)
- Log Penghapusan

Searching:
- Cari Nama Produk (Linear Search)
- Cari ID Produk (Binary Search)
- Cari Kategori Produk (Linear Search)

Sorting:
- Urutkan ID (Bubble Sort)
- Urutkan Nama (Selection Sort)
- Urutkan Stok (Insertion Sort)

Statistik:
- Total Produk Aktif
- Total Stok Barang
- Rata-rata Stok
- Produk dengan Stok Tertinggi
- Produk dengan Stok Terendah
- Produk dengan Harga Tertinggi
- Produk dengan Harga Terendah

File Handling:
- Load data produk dari file TXT

Struktur Data:
Program menggunakan Array Manual:
```java
int[] id;
String[] nama;
String[] kategori;
int[] stok;
double[] harga;
boolean[] aktif;
```

Algoritma yang digunakan:

| Algoritma | Implementasi |
|------------|------------|
| Linear Search | Cari Nama, Cari Kategori |
| Binary Search | Cari ID |
| Bubble Sort | Sort ID |
| Selection Sort | Sort Nama |
| Insertion Sort | Sort Stok |
| Traversal | Tampilkan Data |
| Counting | Statistik |
| File Handling | Load TXT |

Teknologi:

- GitHub
